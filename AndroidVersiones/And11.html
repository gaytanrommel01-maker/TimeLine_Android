Versión: Android 11

Nivel de API: API 30

Android 11 (cuyo nombre en código interno es Red Velvet Cake). La primera actualización Beta pública se lanzó el 11 de junio y fue compatible con Google Pixel 2 y posteriores, y la actualización completa se lanzó el 8 de septiembre de 2020. El primer teléfono lanzado con Android 11 Red Velvet Cake en Europa fue el Vivo X51 5G.

Características:
  • Optimiza la gestión de sensores mediante un mayor control de privacidad y nuevas APIs para hardware plegable. Incluye permisos de un solo uso para cámara/micrófono, sensores de movimiento mejorados (giroscopio, acelerómetro, gravedad) y compatibilidad con bisagras
  • Versión ligera (Android 11 Go Edition) para dispositivos de hasta 2 GB de RAM, ofrece un inicio de aplicaciones hasta un 20% más rapido
  • Integración de nuevo centro de control de electrodomésticos conectados. (Se podrá controlar cualquier aparato electrónico conectado a la red de casa desde un acceso directo de la nueva interfaz).
  • Mejoras de conectividad y experiencia en el uso de redes 5G.
