Versión: Android 2.3

Nivel de API: API 9

Android 2.3 Gingerbread es una versión de Android lanzada el 6 de diciembre de 2010. Android 2.3 se basa en el kernel Linux 2.6.35, que es una corrección de errores de versiones anteriores 2.6.

Características:
  • Diseño de interfaz de usuario actualizado con mayor simplicidad y velocidad
  • Admite varias cámaras en el dispositivo, incluida una cámara frontal, si está disponible.
  • Compatibilidad con tamaños y resoluciones de pantalla extra grandes (WXGA y superiores)
  • Soporte nativo para más sensores (como giroscopios y barómetros)
  • Administración de energía mejorada con un rol más activo en la gestión de aplicaciones que mantienen el dispositivo activo durante demasiado tiempo
