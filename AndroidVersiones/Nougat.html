Versión: Android 7.0

Nivel de API: API 24

Android 7.0 Nougat (no Nogalid) es una versión de Android lanzada oficimente por Google el 22 de Agostso de 2016, el primer dispositivo con el SO preinstalado fue el LG V20. Fue recibido con una gran cantidad de críticas positivas, debido a sus mejoras: multiventana, Vulka, Doze mejorado, etc.

Características:
  • Permite utilizar 2 aplicaciones a la vez con pantalla dividida (multiventana)
  • Mejoras en el modo Doze para el ahorro de batería
  • Soporte para Vulkan, API para mejorar el rendimiento gráfico y soporte para Realidad Virtual
