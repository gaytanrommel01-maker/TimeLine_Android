Versión: Android 9.0

Nivel de API: API 28

Fue lanzado en agosto de 2018 y esta enfocada en la inteligencia artificial, la optimización de la batería ("batería inteligente") y un nuevo sistema de navegación por gestos.

Características:
  • Utiliza machine learning para priorizar aplicaciones y optimizar el rendimiento del sistema.
  • Mayor control sobre los permisos de micrófono y cámara.
  • Introducción de una nueva interfaz de navegación por gestos.
  • Optimiza el uso de sensores de movimiento, ambientales y de posición para mejorar la eficiencia energética, permitiendo una rotación de pantalla inteligente, mejor control de biometría y una gestión del tiempo de uso (Digital Wellbeing) que silencia el móvil al ponerlo boca abajo
