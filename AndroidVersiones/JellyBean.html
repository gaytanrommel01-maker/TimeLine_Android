Versión: Android 4.1 Jelly Bean

Nivel de API: API 16

Android 4.1 Jelly Bean es una versión de Android que se presentó durante Google I/O 2012.

Características:
  • Con Jelly Bean, los usuarios ciegos pueden usar el "Modo Gestos" para navegar de manera confiable por la interfaz de usuario usando gestos táctiles y de deslizamiento en combinación con salida de voz.
  • Mejoras en el rendimiento general del sistema (vsync, triple almacenamiento en búfer, aumento de entrada de CPU para evitar latencia, etc)
  • Mayor compatibilidad con texto bidireccional y otros idiomas
  • Mayor soporte para una amplia gama de sensores, enfocada en la precisión y el bajo consumo energético
