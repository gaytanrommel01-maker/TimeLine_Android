Versión: Android 2.2

Nivel de API: API 8

Android 2.2 (también conocido como Froyo , abreviatura de Frozen Yogurt ) es una versión de Android lanzada el 20 de mayo de 2010. Android 2.2 se basa en el kernel de Linux 2.6.32, que es una corrección de errores de versiones 2.6 anteriores, incluido el kernel 2.6.29 utilizado para Android 2.0/2.1 Eclair.

Características:
  • Optimizaciones de velocidad, memoria y rendimiento
  • Compatibilidad con el servicio Android Cloud to Device Messaging (C2DM), que permite notificaciones push
  • Mejoras adicionales en la velocidad de la aplicación, implementadas mediante compilación JIT
  • Sentó las bases para el uso moderno de sensores en smartphones, admitiendo tres categorías principales: movimiento (acelerómetro), posición (magnetómetro/brújula) y ambientales (luz, proximidad).
  • Compatibilidad con pantallas de alta resolución, como pantallas de 720p de cuatro pulgadas
