Versión: Android 3.0

Nivel de API: API 9

Android 3.0 Honeycomb es una versión de Android lanzada el 22 de febrero de 2011, fue la primera versión diseñada exclusivamente para tabletas. El primer dispositivo con esta versión, la tableta Motorola Xoom, se lanzó el 24 de febrero de 2011.

Características:
  • Nueva IU virtual y “holográfica” diseñada desde cero para uso exclusivo en tablets
  • Optimizó la respuesta a movimientos y orientación, además de añadir soporte para accesorios USB como teclados y ratones, mejorando la interacción con los sensores ambientales y de posición.
  • Soporte para procesadores multinúcleo
  • Aceleración de hardware
